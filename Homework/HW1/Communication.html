MAKING APPS USING JAVA


I'm taking a class here at CSUMB that actually teaches us how to make simple application for Andriod
Devices. The class is CST338 Software Design and the coding language we use to make an app is Java. We also
use an Intergrated Development Enviromeant (or IDE) called Andiod Studio to help with the process. We write
the code in Andriod Studio and some other HTML styles too and then use an Andriod Virtual Device (or AVD) to
test run our app on. The AVD will show up as a smart phone and run the app exactly like it would on a real one.
